# Context — Module Map

## Identity

Responsável por autenticação, sessão e prova de identidade.

Entidades candidatas:
- Identity
- Credential
- Session

## Organizations

Responsável por tenant e estrutura organizacional.

Entidades candidatas:
- Organization
- Tenant

## Users

Responsável por usuário, perfil e memberships.

Entidades candidatas:
- User
- Membership
- UserProfile

## Access Control

Responsável por roles, permissions, assignments e policies.

Entidades candidatas:
- Role
- Permission
- Assignment
- Policy

## Audit Logs

Responsável por eventos auditáveis imutáveis.

Entidades candidatas:
- AuditLogEntry
- AuditActor
- AuditTarget
